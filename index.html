Welcome!

Are you new to JavaScript?

There is a quick and fun tutorial called JS For Cats.

If you're interested in how specific language features and types work the Mozilla Developer Network has excellent documentation on JavaScript.

If you're comfortable using your computer's terminal application then you may want to install Node and work your way through the javascripting workshopper.

Are you new to Node?

The best guide to learning Node once you have some basic JavaScript knowledge is "Art of Node". The guide not only introduces many fundamental concepts but will also send you to various tutorials and other important resources.

Once you're comfortable with Node you may want to learn a bit more about some of the most widely accepted practices which are covered well in the Node Handbook.

Would you like to be more involved?

You can give feedback and contribute to this welcome page.

You can attend or event start a local chapter of Node School to help educate and grow your local community.

The Node Forward website has a list of other projects you may want to get involved in as well.
